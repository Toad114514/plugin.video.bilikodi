# 简介
神秘的**非第三方**b站客户端的 Kodi 20.2 实现  
允许你在kodi上任意浏览b站上的视频（目前在 Nexus 20 测试通过）  
**为了API返回稳定性，bro需要扫码登录你的b站账号！否则不能看**
# 特性
 - 用户登录
 - 主页推荐
   - 支持跳转到Up主或查看详细信息
 - 视频查看
   - 支持最高 720P 视频
   - 支持点赞（测试）
 - 跳转到（目前仅用户）
 - 搜索视频 (停用)
 - 用户/Up主信息
   - 投稿视频
   - 收藏夹查看
   - 用户关注
 - 加密风控
   - wbi_key
   - cookies 刷新(未测试，没有正式启用)
