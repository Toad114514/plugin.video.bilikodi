# Bilikodi
关于我目前找不到kodi上的b站客户端于是我决定自己手搓这件事
# 简介
一个简单的b站客户端插件，允许你在kodi上关注你喜欢的Up主、任意浏览b站上的视频（目前在 Nexus 20 测试通过）
# 特性
 - 用户登录
 - 主页推荐
   - 支持跳转到Up主或查看详细信息
 - 跳转到（目前仅用户）
 - 搜索视频 (停用)
 - 用户/Up主信息
   - 投稿视频
   - 用户关注
 - 加密风控
   - wbi_key
   - cookies 刷新(未测试，没有正式启用)
