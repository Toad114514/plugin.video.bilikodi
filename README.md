# 简介
一个简单的b站客户端插件，允许你在kodi上关注你喜欢的Up主、任意浏览b站上的视频（目前在 Nexus 20 测试通过）
# 特性
 - 主页推荐
 - 跳转到（目前仅用户）
 - 搜索视频
 - Up主信息
   - 投稿视频
   - 合集视频
 - 本地关注列表（灵感自 [InfinityLoop1308/PipePipe](https://github.com/InfinityLoop1308/PipePipe)）
